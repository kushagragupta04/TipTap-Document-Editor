# Tiptap Editor

A professional-grade document editor expressly designed for legal accuracy and precision. Tiptap Editor combines the ease of modern text editing with the strict formatting requirements of the legal profession.

## Key Features

- **Real-Time Pagination Visualization**: See exactly where your pages end while you type, ensuring no surprises during printing.
- **USCIS-Compliant Formatting**: Pre-configured with 1-inch margins and standard legal layouts.
- **Advanced Table Handling**: Create and edit complex tables that split intelligently across pages without breaking data.
- **Seamless PDF Export & Printing**: What you see is truly what you get. Our print engine honors your layout perfectly.
- **Rich Text Formatting**: Full support for standard formatting including **Bold**, *Italics*, <u>Underline</u>, ~~Strikethrough~~, Lists, and Blockquotes.

## Getting Started

To get the editor running locally:

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Start the Development Server**
    ```bash
    npm run dev
    ```

## Usage

Once the application is running, you can:

*   **Edit Content**: Just type as you would in any word processor.
*   **Insert Elements**: Use the toolbar to add tables, images, or lists.
*   **Print/Export**: Click the print icon in the toolbar or use standard browser print shortcuts (Ctrl+P / Cmd+P) to generate a perfectly formatted PDF.
